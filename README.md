# AquaFizz-Employee-Compliance-and-Sales-Analysis

Analyzed over 8,000 employee time logs and 12,000+ sales records using Excel to assess compliance with working hours and evaluate sales performance.

Identified 15%+ non-compliance cases by applying formulas, conditional formatting, and time-based logic to shift data.

Built interactive Excel dashboards using pivot tables, slicers, and charts to visualize compliance trends, regional sales, and top-performing employees.

Provided actionable insights that linked higher compliance rates to a 10% increase in sales conversion, supporting HR and sales optimization strategies.
